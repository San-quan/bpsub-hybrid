# 🚀 开始使用 - 从这里开始！

## ⚡ 3 步启动

### 第 1 步：进入目录
```bash
cd telegram-bot
```

### 第 2 步：启动服务
```bash
./start.sh
```

### 第 3 步：测试机器人
在 Telegram 中：
1. 搜索你的机器人（根据 @BotFather 给的 username）
2. 发送 `/start`
3. 发送测试消息

✅ **配置已就绪，无需修改任何文件！**

## 💬 快速命令

### 用户命令（Telegram 私聊）
```
/start  - 开始使用
/help   - 帮助信息
/faq    - 常见问题
```

### 客服命令（客服群组）
```
/open   - 查看未处理工单
/close  - 关闭工单
/id     - 查看群组 ID
```

### 运维命令（终端）
```bash
./start.sh              # 启动服务
./stop.sh               # 停止服务
./test.sh               # 检查状态
docker-compose logs -f  # 查看日志
```

## 🔧 遇到问题？

1. **查看日志**
   ```bash
   docker-compose logs -f bot
   ```

2. **重启服务**
   ```bash
   docker-compose restart bot
   ```

3. **运行诊断**
   ```bash
   ./test.sh
   ```

4. **查看文档**
   - `使用指南.md` - 故障排查章节
